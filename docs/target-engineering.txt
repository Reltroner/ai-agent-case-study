Kita jadikan proyek ini sebagai **AI Agent Engineering Snapshot**, bukan sekadar “belajar AI agent”. Fondasinya sudah jelas dari dokumen: proyek ini adalah **self-built agentic workflow simulation** untuk membantu bisnis jasa kecil mengubah lead yang tercecer menjadi follow-up yang terstruktur, terprioritaskan, dan tetap melalui human review. 

## Target Akhir Snapshot

Target akhir proyek ini:

> **Membangun snapshot engineering yang membuktikan bahwa kamu bisa merancang AI-assisted workflow dari Problem → Solution → Result, lengkap dengan agent role, data contract, scoring logic, prompt versioning, QA guard, human-in-the-loop, dan dokumentasi portfolio.**

Bukan target akhir:

```text
Membuat SaaS CRM full production.
Membuat WhatsApp bot otomatis.
Membuat dashboard kompleks.
Membuat AI agent autonomous tanpa review manusia.
```

Target akhir yang realistis:

```text
AI Lead Follow-Up Assistant
= agentic simulation prototype
= lead input -> AI processing -> structured output -> human review
```

# 1. Final Snapshot yang Harus Jadi

Pada akhir proyek, snapshot harus punya 7 bukti utama:

## 1. Business Problem Locked

```text
Small service businesses lose potential clients because incoming leads are scattered, follow-up is manual, and serious prospects are not prioritized.
```

## 2. Workflow Design

```text
Lead Input
-> Lead Intake Agent
-> Lead Qualification Agent
-> Need Extraction Agent
-> Follow-Up Strategy Agent
-> Message Drafting Agent
-> CRM Update Agent
-> QA Guard Agent
-> Human Review
```

## 3. Data Contract

Output agent harus punya JSON standar:

```json
{
  "lead_id": "",
  "lead_category": "hot | warm | cold",
  "lead_score": 0,
  "main_need": "",
  "missing_information": [],
  "next_best_action": "",
  "follow_up_message": "",
  "crm_status": "",
  "next_follow_up_date": "",
  "risk_flags": [],
  "requires_human_review": true
}
```

## 4. Sample Dataset

Minimal:

```text
10 dummy leads
```

Dengan variasi:

```text
hot lead
warm lead
cold lead
budget unclear
timeline unclear
service unclear
urgent lead
low-intent lead
duplicate-like lead
incomplete lead
```

## 5. Manual Simulation Output

Minimal:

```text
10 leads berhasil diproses manual
10 output JSON tersimpan
Setiap output punya classification, score, draft, next action, dan human review flag
```

## 6. Semi-Automated Prototype

Versi berikutnya:

```text
Google Form / manual trigger
-> n8n
-> AI processing
-> parse JSON
-> Google Sheet
-> owner notification
```

## 7. Portfolio Case Study

Akhirnya harus bisa dijelaskan sebagai:

```text
Problem:
Lead follow-up bisnis jasa kecil masih manual dan tercecer.

Solution:
AI-assisted workflow untuk klasifikasi lead, ekstraksi kebutuhan, draft follow-up, dan CRM status.

Result:
Lead menjadi lebih rapi, prioritas lebih jelas, draft lebih cepat dibuat, dan owner tetap memegang kontrol sebelum pesan dikirim.
```

Isi minimal:

```md
# AI Lead Follow-Up Assistant — Project Framing

## Project Type
Self-built AI agentic workflow simulation prototype.

## Target User
Small service business owners who receive leads from Instagram, WhatsApp, website forms, or referrals.

## Problem
Small service businesses often receive leads from multiple channels, but follow-up is still manual, scattered, and inconsistent.

## Business Impact
Serious prospects may be missed, owners repeatedly write similar replies from scratch, and there is no clear system for prioritizing leads.

## Solution
AI Lead Follow-Up Assistant simulates an AI-assisted workflow that captures lead information, classifies priority, extracts customer needs, generates follow-up drafts, updates CRM status, and keeps human review before sending.

## Result Goal
The project demonstrates how scattered lead inquiries can become structured, prioritized, and ready-to-review follow-up actions.

## Safety Principle
The AI does not send messages automatically. Every draft requires human review.

## MVP Scope
- 10 sample leads
- Lead classification
- Lead scoring
- Need extraction
- Follow-up draft
- CRM status
- Human review flag
- JSON output
- Manual simulation
- Case study

## Out of Scope
- WhatsApp API
- Instagram DM integration
- Full CRM
- Payment system
- Multi-user dashboard
- Auto-send behavior
- Real production deployment
```
